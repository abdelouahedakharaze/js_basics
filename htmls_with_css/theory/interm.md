# Level 2: Intermediate Portfolio with Grid

A responsive portfolio using CSS Grid and advanced styling.

## **New HTML Tags**
| Tag            | Purpose                              |
|----------------|--------------------------------------|
| `<article>`    | Self-contained content               |
| `<nav>`        | Navigation links                     |
| `<div>`        | Generic containers (styled with CSS) |
| `<span>`       | Inline text grouping                 |
| `<link>`       | Imports Google Fonts                 |

## **New CSS Properties**
| Property            | Purpose                                  |
|---------------------|------------------------------------------|
| `display: grid`     | Grid layout for projects                 |
| `grid-template-columns` | Responsive columns (`repeat(auto-fill)`) |
| `gap`               | Grid item spacing                       |
| `flex`              | Flexbox for profile card                 |
| `object-fit`        | Image cropping (cover)                  |
| `transition`        | Hover animations                        |
| `:hover`            | Interactive states                      |
| `position: sticky`  | Fixed header                            |
| `::before`/`::after`| Pseudo-elements                         |
| `@media`            | Responsive breakpoints                  |

## **Key Features**
- **CSS Grid** for project layouts  
- **Flexbox** for profile card alignment  
- **Google Fonts** integration  
- **Hover effects** on cards/links  
- **Media queries** for mobile responsiveness
